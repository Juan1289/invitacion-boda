# 💍 Nuestra Boda | Fabián & Luz Aida

Sitio web interactivo de invitación de boda desarrollado con HTML, TailwindCSS y JavaScript.

La página ofrece una experiencia elegante y moderna para invitados, incluyendo contador regresivo, música de fondo, navegación interactiva, confirmación de asistencia y dedicatorias personalizadas.

## ✨ Características

- 💍 Diseño elegante y responsive
- 🎵 Música de fondo interactiva
- ⏳ Contador regresivo para la boda
- 📱 Navegación táctil y animaciones suaves
- 🎠 Transiciones dinámicas entre secciones
- 📊 Gráfico interactivo con Chart.js
- 💌 Confirmación vía WhatsApp
- 🤖 Generador de dedicatorias con IA
- 🎨 Diseño visual premium con TailwindCSS

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- TailwindCSS
- Chart.js
- Gemini API
